# 📧 Sheet Mail Merge 使用指南

> 🚀 在 Google Sheet 中一鍵寄送客製化 Email（類似 YAMM）

---

# 🧭 產品介紹

**Sheet Mail Merge** 是一個整合：

* Google Sheet（資料來源）
* Gmail 草稿（信件範本）

的自動化郵件工具。

你可以透過 `{{欄位}}` 設計信件，系統會自動帶入資料並寄送。

---

# ⚡ 快速開始（3 分鐘上手）

## ① 建立 Gmail 草稿（信件範本）

請先在 Gmail 建立一封草稿，例如：

```
親愛的 {{姓名}} 您好，

我們是 {{公司名稱}}，
這封信是專門為您準備的通知。

謝謝！
```

### 📌 規則

* 必須使用 `{{欄位名稱}}`
* 欄位名稱需與 Sheet 第一列完全一致

---

## ② 準備 Google Sheet

在第一列建立欄位，例如：

| 收件者 | 副本 | 附件 | 姓名 | 公司名稱 |
| --- | -- | -- | -- | ---- |

---

## ③ 啟動 Mail Merge

1. 開啟 Google Sheet
2. 點擊 Chrome 擴充功能
3. 開啟 Mail Merge UI

---

# 🧩 系統欄位說明

系統會使用以下欄位（可自動建立）：

| 欄位名稱   | 用途       |
| ------ | -------- |
| 收件者    | Email    |
| 副本     | CC       |
| 附件     | Drive 連結 |
| 郵寄狀態   | 寄送狀態     |
| 狀態更新時間 | 最後更新     |
| 執行紀錄   | 詳細紀錄     |

👉 若缺少，點擊「初始化」即可自動建立

---

# 🔄 使用流程

## Step 1：初始化

* 初次使用 → 自動建立欄位
* 已有欄位 → 直接進入

---

## Step 2：選擇草稿

系統會自動顯示：

✅ **只包含 `{{欄位}}` 的草稿**

---

## Step 3：預覽

* 使用最新一筆資料
* 即時替換 `{{欄位}}`
* 顯示 HTML 信件內容

---

## Step 4：寄送

### 📬 立即寄送

* 馬上寄出
* 更新狀態為 `SENT`

### ⏰ 排程寄送

* 設定時間
* 自動寄出
* 狀態為 `SCHEDULED`

---

# 📊 郵寄狀態說明

| 狀態        | 說明   |
| --------- | ---- |
| PENDING   | 準備寄送 |
| SCHEDULED | 已排程  |
| SENT      | 已寄出  |
| FAILED    | 失敗   |

---

# 📎 附件規則

* 使用 Google Drive 連結
* 多附件 → 換行分隔
* 任一錯誤 → 整筆失敗

---

# ⚠️ 注意事項

## 1️⃣ 欄位名稱需完全一致

```
{{姓名}} ✅
{{名字}} ❌
```

---

## 2️⃣ 防止重複寄送

系統只會寄送：

👉 郵寄狀態為「空白」的資料

---

## 3️⃣ 帳號一致性

請確認：

* Gmail
* Google Sheet
* Mail Merge

👉 使用同一個 Google 帳號

---

# 💡 常見問題

## Q：為什麼看不到草稿？

👉 草稿沒有 `{{欄位}}`

---

## Q：為什麼寄送失敗？

👉 常見原因：

* Email 格式錯誤
* 附件連結無效
* 缺少欄位

---

## Q：可以重寄嗎？

👉 將「郵寄狀態」清空即可

---
